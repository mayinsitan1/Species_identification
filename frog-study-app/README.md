# 无尾两栖类速学网页应用

打开地址：

- 本机：http://127.0.0.1:8765/frog-study-app/
- 同一 Wi-Fi 下手机：http://192.168.8.111:8765/frog-study-app/

功能：

- 按科筛选、关键词搜索物种
- 按广东分布筛选：在广东有分布 / 在广东无分布
- “在广东有分布”除直接出现“广东”外，也会纳入全国广布、大部分省、长江以南、华南、中国南部、从云南到浙江等明显覆盖广东的描述
- 查看中文名、拉丁名、科属分类、鉴别特征、分布、生境、习性
- 查看照片、播放鸣声
- 收藏物种、标记已掌握
- 随机小测：可选择“听鸣声辨认”或“看图片辨认”，题库跟随当前地区筛选和科筛选；同一物种的多个资源会以“鸣声段落 1”“图片 1”等匿名标签展示

数据来源于 `../NSII_无尾两栖类鸣声专题`。网页应用本体约 9.4 MB，照片和鸣声原始资源仍保留在原数据目录中。
